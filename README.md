# DFM Intelligence Platform

OEE Executive Dashboard powered by Google Sheet `ALL_DATA`.

## Deploy
1. Upload all files to GitHub repository `DFM-Intelligence`.
2. Import repository to Vercel.
3. Framework: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.

## Google Sheet
The dashboard reads data from published CSV using gid `1464551640`.
Make sure the Google Sheet permission is: `Anyone with the link -> Viewer`.
